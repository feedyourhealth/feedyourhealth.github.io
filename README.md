# 🥗 Dietologist - Εργαλείο Διαιτολόγου

**Μια ολοκληρωμένη, single-page εφαρμογή για διαχείριση διατροφικών σχεδίων με περισσότερες από 300 τροφές.**

## ✨ Χαρακτηριστικά

- 📋 **Δημιουργία εβδομαδιαίων πλάνων** για πολλαπλούς πελάτες
- 🔄 **Αποθήκευση δεδομένων τοπικά** στο browser (localStorage)
- 🧮 **Υπολογισμοί θερμίδων & μακροθρεπτικών** σε real-time
- 📊 **Ανάλυση Mediterranean Diet score** για κάθε σχέδιο
- 🎯 **Εξατομικευμένοι στόχοι** ανά μέρα ή ανά πελάτη
- 📖 **Αναλυτικές συνταγές** από τον chef Άκης Πετρετζάκης
- 🖨️ **Εξαγωγή PDF** - Τα συστατικά των συνταγών εμφανίζονται αυτόματα στο έγγραφο
- 📱 **Responsive Design** - Δουλεύει σε όλες τις συσκευές

## 🎨 Δομή τροφών

Κάθε τρόφιμο περιλαμβάνει:
- **Kcal** - Θερμίδες
- **P** - Πρωτείνες (g)
- **C** - Υδατάνθρακες (g)
- **F** - Λίπη (g)
- **Fi** - Φυτικές ίνες (g)

## 👨‍🍳 Chef Recipes - Άκης Πετρετζάκης

- **High Protein Ομελέτα Wrap**: 850 kcal | 60g protein
- **Wrap με τονοσαλάτα**: 564 kcal | 45g protein

*Κάθε συνταγή περιλαμβάνει πλήρη λίστα συστατικών που εμφανίζεται κατά την εκτύπωση.*

## 🚀 Χρήση

1. Ανοίξτε το [index.html](https://feedyourhealth.github.io) σε browser
2. Δημιουργήστε νέο πελάτη
3. Ορίστε προσωπικές παραμέτρους (ηλικία, φύλο, BMR, κλπ)
4. Δημιουργήστε ή επιλέξτε πρότυπο πλάνου
5. Προσθέστε τρόφιμα χρησιμοποιώντας drag & drop ή αναζήτηση
6. Εκτυπώστε ή εξάγετε το σχέδιο ως PDF

## 💾 Δεδομένα

Όλα τα δεδομένα αποθηκεύονται **τοπικά στον browser** σας. Δεν αποστέλλονται σε διακομιστή.

## 📚 Βάση Δεδομένων
